Null Object Pattern for PSR-11 Container

## Requirements

The following versions of PHP are supported: `7.4`, `8.0`, `8.1`.

## Install

```bash
$ composer require brenoroosevelt/psr-nullcontainer
```

## Usage
```php
$container = new \BrenoRoosevelt\Psr11\NullContainer();
```
